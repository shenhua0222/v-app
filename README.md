## 技术栈
 
 - vue-cli
 - vue2.0
 - vue-router
 - axios
 - less 
 
## 实现功能
 
 1. 用户登录
 2. 下拉加载
 3. 主题发布
 4. 主题收藏
 5.  回复评论
 6. 个人中心
 
## 执行项目
 
``` bash
# install dependencies
npm install
 
# serve with hot reload at localhost:8080
npm run dev
 
 
```
 
## 总结

 - 没用ui框架，界面不够美观
 - bottom组件中，绑定style时，用原生进行了dom操作
 - 经测试，axios兼容ie，消耗了部分性能
 - 时间原因，暂未移动适配；
